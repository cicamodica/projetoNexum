## Estrutura
As telas seguem o projeto de interface elaborado anteriormente, os templates criados estão disponíveis aqui e são composto pelos seguintes layouts:

- Tela Homepage do usuário não logado
- Tela Cadastro
- Tela de Login
- Tela Homepage da ONG
- Tela Homepage do ADMIN
- Tela de Marketplace (Vitrine)
- Tela de Marketplace (Modal doação de material)
- Tela de Marketplace (Modal doação de valor)
- Tela de Marketplace (Modal de vaga)
- Tela de Perfil ONG
- Tela Fale Conosco
- Tela Esqueceu a senha?
- Tela de ONGs cadastradas
- Tela Resultados de Pesquisa

### Tela Homepage do usuário não logado
Tela onde o usuário que não possui cadastro pode pesquisar por vagas ou doações, além de buscar por ONGs específicas.


### Tela Cadastro
Tela apresenta campos para preenchimento de dados para cadastro de usuários (ONGs). O usuário deve preencher os dados da ONG como: Razão social, CNPJ, Endereço, Descrição de atividades, E-mail etc. e deve adicionar uma imagem de perfil e o documento que comprova e existência da ONG. Todos os campos são obrigatórios.


### Tela de Login
Tela onde o usuário cadastrado pode logar no sistema, inserindo E-mail para login e senha para autenticação, exibindo as opções "Esqueceu a senha?" e "Não possui cadastro?", que redirecionam para outras páginas.
